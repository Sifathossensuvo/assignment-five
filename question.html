What is the document object model?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each element, attribute, and text is a node. The DOM allows programming languages like JavaScript to interact with and manipulate the content, structure, and styles of a webpage dynamically. It enables developers to add, modify, or remove elements and respond to user interactions.

How do you select an element from the dom?

To select an element from the DOM, use document.getElementById('id') for selecting by ID. For class selection, use document.querySelector('.className'). If you want to select all matching elements, use document.querySelectorAll('.className'). You can also select elements by tag name using document.getElementsByTagName('tag').

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element instead of multiple listeners on individual child elements. When an event occurs on a child element, it bubbles up to the parent, allowing the parent to handle the event. This approach is useful because it improves performance by reducing the number of event listeners and makes it easier to manage dynamic content, as new child elements will automatically inherit the parent's event listener without needing to add more listeners.

How do you manipulate an elements attributes and styles using the dom?

element.setAttribute('attributeName', 'value') to set an attribute and element.getAttribute('attributeName') to retrieve its value. For styles, you can modify inline styles by accessing the element.style property, such as element.style.backgroundColor = 'blue' to change the background color. You can also directly modify specific attributes like element.id or element.className. These methods enable dynamic changes to an element's attributes and styles based on user interactions.
